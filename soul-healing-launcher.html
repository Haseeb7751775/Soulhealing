How the connection works
1
Open the Client Site in one tab and the Admin Panel in another — both must be in the same browser.
2
Both tabs share a live sync channel — data flows instantly between client and admin in real time, with no server required.
3
When a client submits a form, books a session, or chats, data is written to localStorage and broadcast instantly to the admin tab — no server required.
4
The admin can reply to messages, assign time slots, and update payment details — all changes appear on the client side in real time.
5
Important: All three files must be opened from the same folder or served from the same local server for storage to be shared.
